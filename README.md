# Password Strength Checker
Tool CLI kiểm tra độ mạnh password theo 7 tiêu chí bảo mật. Hỗ trợ check single password hoặc batch check từ file, xuất kết quả ra CSV.

## Features
- Check 7 tiêu chí: length, uppercase, lowercase, digit, special char, common patterns, whitespace
- Scoring system: Weak / Medium / Strong / Very Strong
- Batch check từ file text
- Export kết quả ra CSV cho reporting

## Tech Stack
- Python 3.10+
- Standard library only (no external dêpndencies)
